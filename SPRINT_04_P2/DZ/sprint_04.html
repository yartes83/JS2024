Task 03

Функция должна выводить прямоугольник на страницу состоящий из символов звездочка.

В горизонтали 6 звездочек, количество линий - число, которое пользователь ввел в input.i-3.

Например для введенного числа 3 должно получится так:

******
******
******

Перенос строки делайте с помощью тега br.

input i-3

Task 04

Функция должна выводить прямоугольник на страницу состоящий из чисел. В горизонтали 6 чисел от 5 до нуля, количество линий - число, которое пользователь ввел в input.i-4.

Например для введенного числа 3 должно получится так:

5_4_3_2_1_0_
5_4_3_2_1_0_
5_4_3_2_1_0_

Перенос строки делайте с помощью тега br.

input i-4

Например, для введенных чисел 4 и 2:

4_3_2_1_0_

Перенос строки делайте с помощью тега br.
